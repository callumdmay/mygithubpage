Raspberry Pi Notification Center

This is a hobbyist notification app that I made for the raspberry pi that utilizes the pi touchscreen. It is primarily written in AngularJS, although there is a separate python web server that serves the icloud reminder and a service that turns off the display when the light level in the room is low (at night). Further details and source code can be found on Github

  • Language: Javascript
